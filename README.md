# Quote2PDF Application

**Quote2PDF** is an application designed for skilled tradesmen to create professional quotations and generate PDF files that can be easily shared with clients via WhatsApp.  

Over time, this project will evolve into a comprehensive **business management SaaS** tailored for trade-based businesses.  

---

## Features

Currently, the application focuses on:  

- Creating and managing quotations  
- Generating PDF files from quotations  
- Sharing PDFs with clients  

Planned future features include:  

- Dashboard with key business metrics  
- User authentication and authorization  
- Database storage for quotations  
- Email integration for sending quotes  
- SMS integration for notifications  
- Payment integration  
- Mobile app for on-the-go access  
- Project and task tracking  
- Customer management  
- Invoice management  
- Business settings for SaaS tenants  

---

## Technology Stack

**Frontend:**  
- React.js  
- Tailwind CSS  

**Backend:**  
- Node.js  
- Express.js  
- Puppeteer (for PDF generation)  
- EJS (template engine)  
- PDFKit (alternative PDF generation)  

**Database:**  
- PostgreSQL

## Getting Started

1. **Clone the repository**  
   ```bash
   git clone <repository-url>
   cd Quote2PDF

2. **Install dependencies**  
   ```bash
   npm install

3. **Set up environment variables**  
   ```bash
   cp .env.example .env
   # Edit .env file with your database and other configuration settings

4. **Run the application**  
   ```bash
   npm run dev

5. **Access the application**  
   Open your browser and navigate to `http://localhost:3000` to access the application.
